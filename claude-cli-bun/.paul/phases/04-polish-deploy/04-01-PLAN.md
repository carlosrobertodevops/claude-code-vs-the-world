## Goal
Polir a camada de UX global: `ThemeProvider` + dark mode, sidebar responsiva (drawer mobile, fixa desktop), header com user menu + toggle de tema, background com linhas horizontais/verticais finas e blur usando `public/backgroud.png`, componentes `EmptyState` e `PageHeader`, tipografia Inter, e `RevenueChart` (Recharts) na home do dashboard. Revisar 375px e dark mode em todas as páginas.

## Purpose
Entregar a experiência visual consistente que o spec pede — hoje temos só layout cru de sidebar+content. Sem isso, o produto "parece" incompleto mesmo com a API pronta.

## Output
- `src/components/layout/*` com 7 componentes reutilizáveis
- `ThemeProvider` (next-themes) + toggle dark/light no header
- Background visual aplicado globalmente (linhas finas H+V + blur sobre `backgroud.png`)
- Inter aplicada via `next/font/google`
- Dashboard home com KPIs em cards + `RevenueChart` (LineChart do Recharts)
- Responsividade mobile (375px) verificada em `/login`, `/dashboard`, `/dashboard/clientes`, `/dashboard/servicos`, `/fila/:slug`
</objective>
