## AC-1: docker-compose.prod.yml válido

```gherkin
Given docker-compose.yml (dev) como baseline
When cria docker-compose.prod.yml
Then prod compose tem:
  - app com restart: unless-stopped e image (pullable) em vez de build local (opcional — pode manter build)
  - postgres com volumes nomeados e restart policy
  - SEM minio (prod usa S3/MinIO externo via env)
  - SEM bind mounts (apenas named volumes)
  - `docker compose -f docker-compose.prod.yml config` retorna YAML válido
```
