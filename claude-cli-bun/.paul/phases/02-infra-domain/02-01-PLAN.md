  <files>drizzle/schema/*.ts, drizzle/schema/index.ts</files>
  <action>
    Criar schemas Drizzle seguindo o spec em planejamento-bun.md:
    - **users**: id (cuid), name, email (unique), password_hash, role (enum MANAGER/EMPLOYEE), timestamps, deleted_at
    - **customers**: id, name, cpf/cnpj, phone, email, notes, timestamps, deleted_at
    - **vehicles**: id, customer_id (FK), plate, make, model, year, color, timestamps, deleted_at
    - **vehicle_photos**: id, vehicle_id (FK), upload_id (FK file_uploads), timestamps
    - **products**: id, name, sku, unit, price, stock_qty, min_stock, timestamps, deleted_at
    - **stock_movements**: id, product_id, qty_delta, reason, ref_service_order_id (nullable), created_at, user_id
    - **service_types**: id, name, description, base_price, duration_minutes, timestamps
    - **quotes**: id, customer_id, vehicle_id, status (DRAFT/SENT/ACCEPTED/REJECTED), total, notes, timestamps
    - **quote_items**: id, quote_id, service_type_id (nullable), product_id (nullable), description, qty, unit_price
    - **contracts**: id, customer_id, quote_id (nullable), status (PENDING/SIGNED/CANCELLED), signed_at, signature_token, body, timestamps, deleted_at
    - **service_orders**: id, customer_id, vehicle_id, employee_id (FK users), status (enum PENDING/IN_PROGRESS/DONE/DELIVERED/CANCELLED), notes, total, started_at, finished_at, timestamps, deleted_at
    - **service_order_items**: id, service_order_id, service_type_id (nullable), product_id (nullable), qty, unit_price
    - **service_photos**: id, service_order_id, upload_id
    - **queue_entries**: id, service_order_id, slug (FK car_wash_config.slug), position, status (enum WAITING/IN_SERVICE/DONE), timestamps
    - **car_wash_config**: id, slug (unique), name, open_hours (jsonb), created_at, updated_at
    - **file_uploads**: id, storage_key, mime, size, uploaded_by, created_at

    Usar `pgTable`, `text`, `varchar`, `integer`, `decimal`, `boolean`, `timestamp`, `jsonb`, `pgEnum` de drizzle-orm/pg-core.
    PK: `text("id").primaryKey().$defaultFn(() => createId())` (cuid2).
    Relações via `relations()` ou FKs diretos.
    `drizzle/schema/index.ts` exporta tudo (schemas + enums + relations).

    Avoid:
    - PK integer.
    - `timestamp without time zone` — usar `timestamp({ withTimezone: true })`.
    - Campos que spec não menciona (scope creep).
  </action>

    Criar `src/server/infrastructure/drizzle/client.ts`:
    ```ts
    import { drizzle } from "drizzle-orm/postgres-js";
    import postgres from "postgres";
    import * as schema from "../../../../drizzle/schema";

    const connectionString = process.env.DATABASE_URL!;
    const client = postgres(connectionString, { max: 10 });
    export const db = drizzle(client, { schema });
    export type DB = typeof db;
    ```

    Avoid:
