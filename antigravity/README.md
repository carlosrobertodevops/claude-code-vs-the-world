# 🌊 AquaWash

**Sistema de Gestão para Lava-Jatos** — Micro-SaaS completo.

## Features Extras
- 🌙 **Dark Mode** — Conforto visual em qualquer ambiente
- 💬 **WhatsApp Integration** — Comunicação direta com clientes
- 📥 **Exportação CSV** — Relatórios exportáveis

## Quick Start

```bash
npm install
docker compose up -d postgres
npx prisma migrate dev
npx prisma db seed
npm run dev
```
